# 合约交易学习笔记 - 多空双向策略

## 📚 核心概念（2026-03-05）

### 1. 订单块（Order Block）
- **定义**：机构订单聚集区域，价格在此区域反转
- **看涨订单块**：价格快速上涨前的最后一根下跌K线
- **看跌订单块**：价格快速下跌前的最后一根上涨K线
- **多空双向应用**：
  - 跌破看涨订单块 = 做空信号
  - 突破看跌订单块 = 做多信号

### 2. 流动性掠夺（Liquidity Sweep）
- **定义**：价格故意突破关键支撑/阻力，触发止损单后反向运动
- **识别**：假突破 + 快速反向
- **多空双向应用**：
  - 向上假突破 → 做空（掠夺多头流动性）
  - 向下假突破 → 做多（掠夺空头流动性）

### 3. FVG（Fair Value Gap）
- **定义**：价格快速移动留下的不平衡区域
- **看涨FVG**：中间K线低点高于前后K线高点
- **看跌FVG**：中间K线高点低于前后K线低点
- **多空双向应用**：
  - 价格回踩FVG = 入场机会
  - 跌破FVG = 趋势反转信号

### 4. 供需区（Supply/Demand Zones）
- **需求区**：价格快速上涨前的整理区域（做多区）
- **供给区**：价格快速下跌前的整理区域（做空区）
- **多空双向应用**：
  - 需求区被跌破 → 转为供给区 → 做空
  - 供给区被突破 → 转为需求区 → 做多

### 5. 突破策略（Breakout）
- **真突破**：放量突破 + 回踩确认
- **假突破**：缩量突破 + 快速回落
- **多空双向应用**：
  - 真突破 → 顺势追单
  - 假突破 → 反向操作

## 🎯 多空双向交易框架

### 做多条件（全部满足）
1. 价格触及需求区/订单块支撑
2. 出现看涨K线形态（锤子线、吞没等）
3. 美股强势（纳指上涨）
4. 资金费率低/负值（空头拥挤）
5. ETH清算地图显示下方有密集空头清算

### 做空条件（全部满足）
1. 价格触及供给区/订单块阻力
2. 出现看跌K线形态（流星线、吞没等）
3. 美股弱势（纳指下跌）
4. 资金费率高（多头拥挤）
5. ETH清算地图显示上方有密集多头清算

### 风控规则
- 杠杆：5-10x（根据波动率调整）
- 止损：关键支撑/阻力外1-2%
- 止盈：1.5-2倍止损距离
- 仓位：单次不超过总资金10%

## 💡 关键洞察

**多空双向的核心思维**：
- 涨了可以空（找假突破、流动性掠夺）
- 跌了可以多（找支撑位、订单块）
- 不要只做一个方向！
- 市场80%时间是震荡，只有20%是趋势
