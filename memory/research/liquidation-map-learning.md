# 清算地图学习笔记

## 什么是清算地图
清算地图（Liquidation Map）显示不同价格位置的爆仓金额分布。

### 图表解读
- **X轴**: 价格
- **Y轴**: 清算强度/爆仓金额
- **柱子越高**: 该价位爆仓金额越大
- **黄色/橙色柱**: 高清算区 = 价格磁铁

## ETH 当前清算地图分析

### 关键价位
- **当前价格**: 2009.9
- **上方高清算区**: 2050-2100（大量空头会在这里爆仓）
- **下方清算区**: 1950-1900（相对较弱）

### 市场判断
- 价格正在向 2050 的高清算区移动
- 多头趋势明显
- 机构可能利用这个流动性区域入场

### 应该怎么做

- 判断价格会向高清算区移动
- 应该做多而不是做空

## 策略应用

### 清算地图用法
1. **找价格磁铁**: 高清算柱 = 价格容易被吸引过去
2. **突破交易**: 价格突破高清算区后可能加速
3. **止损设置**: 避免把止损放在高清算区（容易被猎杀）
4. **反向操作**: 价格到达高清算区后可能反转
