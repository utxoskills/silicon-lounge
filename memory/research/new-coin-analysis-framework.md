# 新币调研分析方法总结

## 核心原则
**消息面是一切，综合分析后再决策**

---

## 调研步骤（Checklist）

### 1. 基础信息收集
- [ ] **上线时间** - 精确到小时
- [ ] **上线交易所** - 大所(KuCoin/币安)vs小所
- [ ] **代币总量** - 总供应量、流通量
- [ ] **预售情况** - 价格、募集金额、平台

### 2. 链上数据分析
- [ ] **持有者分布** - 前10名持有比例
- [ ] **项目方持仓** - 是否控盘严重(>70%危险)
- [ ] **解锁计划** - 明天解锁多少？抛压大不大？
- [ ] **链上交易活跃度** - 转账次数、活跃地址数
- [ ] **DEX流动性** - Uniswap等是否有池子？价格多少？

### 3. 投资方/合作方核实
- [ ] **真正投资方** - 谁投了？投了多少？
- [ ] **生态合作** - 是投资还是只是合作？(很多项目虚假宣传)
- [ ] **交易所合作** - 是上线还是深度合作？

### 4. 团队背景调查
- [ ] **团队信息** - 官网是否有团队介绍？
- [ ] **LinkedIn** - 团队成员是否真实？
- [ ] **过往项目** - 团队之前做过什么？

### 5. 项目基本面
- [ ] **产品/技术** - 解决什么问题？有实际应用吗？
- [ ] **用户数据** - 有多少真实用户？
- [ ] **收入情况** - 有盈利吗？
- [ ] **竞品分析** - 市场上有没有类似项目？

### 6. 社区热度
- [ ] **Twitter粉丝** - 多少？增长趋势？
- [ ] **互动数据** - 点赞、转发、评论
- [ ] **Discord/Telegram** - 社区活跃度
- [ ] **KOL站台** - 谁推荐？是真的站台还是付费广告？

### 7. 市场环境
- [ ] **大盘趋势** - 比特币走势？市场情绪？
- [ ] **同类项目表现** - 最近上线的类似币表现如何？
- [ ] **资金流入** - 市场有没有钱？

---

## 风险信号（红旗）

**一票否决项：**
- [ ] 项目方控盘 >70%
- [ ] 团队信息不透明/匿名
- [ ] 没有实际产品，纯概念
- [ ] 虚假宣传投资方
- [ ] 解锁抛压巨大（>20%）
- [ ] 小交易所上线
- [ ] 合约代码未审计

**谨慎项：**
- [ ] 前10名持有 >80%
- [ ] 预售价格过高
- [ ] 社区热度低
- [ ] 没有大牌KOL站台

---

## 决策流程

### 狙击（做多）条件：
1. 项目方控盘 <50%
2. 有明确大牌投资方
3. 团队信息透明
4. 有实际产品/用户
5. 上线大所
6. 社区热度高
7. 解锁抛压小

### 打狗（做空）条件：
1. 价格已经暴涨
2. 利好出尽
3. 项目方开始出货
4. 市场情绪过热

### 放弃条件：
- 任一红旗出现
- 信息不透明
- 风险不可控

## 工具清单

- **推特/X** - 找新币消息、KOL观点
- **CoinMarketCap/CoinGecko** - 基础数据
- **Arbiscan/Etherscan** - 链上数据
- **DEX Screener** - DEX价格、流动性
- **Uniswap** - 链上swap交易
- **Tally** - 代币销售平台
- **Pulse** - 新闻聚合
