# PARTI Token (Particle Network) 研究报告

## 项目概述

**PARTI** 是 Particle Network 的原生代币，Particle Network 是一个专注于链抽象（Chain Abstraction）的 Layer-1 区块链。

## 关键信息

### 上市信息
- **上市时间**：2025年3月25日
- **上市交易所**：Binance、OKX、Gate.io
- **Binance HODLer空投**：第13个项目
- **交易对**：USDT、USDC、BNB、FDUSD、TRY

### 代币经济
- **总供应量**：10亿枚
- **HODLer空投分配**：3%
- **营销预算**：3%（上市后6个月开始分配）
- **初始流通**：约23.3%（2.33亿枚）

## 项目分析

### 核心功能
1. **Universal Accounts**：单一账户跨链管理
2. **Universal Liquidity**：统一流动性层
3. **Universal Gas Token**：通用Gas代币
4. **链抽象基础设施**：简化跨链交互

### 技术优势
- 基于 Cosmos SDK 构建
- 双质押模型（Dual Staking）
- 已支持70+条链
- 累计2000万+ UserOps
- 保护资产超过20亿美元

### 融资背景
- **种子轮**：700万美元
- **投资机构**：HashKey Capital、Animoca Brands、GSR Ventures

### 用户数据
- 已创建钱包：1700万+
- 处理交易：1000万+
- 成立时间：2022年4月
