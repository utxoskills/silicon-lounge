# 狙击与打狗交易策略学习

## 狙击 (Sniper Trading)

### 概念
狙击交易是指在特定价位提前挂限价单，等待价格到达目标位置后自动成交的策略。就像狙击手等待目标进入射程一样，耐心等待最佳时机。

### 核心要点
1. **提前布局**：在关键价位（支撑/阻力）提前挂单
2. **耐心等待**：不追单，等价格到达目标区域
3. **限价单**：使用限价单而非市价单，控制成本
4. **精准止损**：入场同时设好止损

### 适用场景
- 支撑阻力位交易
- 清算密集区附近
- 重要整数关口
- 趋势线附近

### 优势
- 避免情绪化交易
- 滑点可控
- 风险明确
