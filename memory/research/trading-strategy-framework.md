# 策略体系形成记录

> **注意：这是交易模块（ETH等主流币合约做多/做空），不是投研模块**

## 核心原则
策略不是固定的，而是**动态组合**，根据市场条件灵活调整。

## 多维度分析框架

### 1. 技术分析维度
- **ICT概念**: BOS/CHoCH结构突破、FVG公允价值缺口、订单块
- **市场结构**: 高低点排列、趋势方向
- **支撑阻力**: 关键价位、供需区

### 2. 链上/数据维度
- **清算图**: 识别高杠杆密集区（价格磁铁）
- **资金费率**: 极端情绪反向信号
- **持仓量**: 趋势确认或背离
- **交易所流入流出**: 资金流向

### 3. 宏观/新闻维度
- **美股联动**: MSTR（BTC领先指标）、纳指（风险情绪）
- **重要数据**: 非农、CPI、利率决议
- **行业动态**: ETF、监管、大额融资
- **地缘政治**: 战争、国际关系

### 4. 时间维度
- **Killzones**: 伦敦开盘、纽约开盘、收盘
- **工作日效应**: 周末流动性、周一方向选择
- **月度效应**: 月初资金流入、季度末调仓

## 策略组合逻辑

### 高概率 setups（多维度共振）
当以下3个以上维度同时满足时，高概率setup：

1. **技术**: 结构突破（BOS）+ FVG未回补
2. **清算**: 价格接近高杠杆清算区
3. **资金费率**: 极端值（>0.1%或<-0.1%）
4. **美股**: MSTR与纳指同向，且ETH/BTC未跟随
5. **时间**: 纽约开盘killzone

### 策略优先级（动态调整）

| 市场环境 | 主导策略 | 辅助工具 |
|---------|---------|---------|
| 趋势明确 | 趋势跟踪 + 结构突破 | 清算图确认目标 |
| 区间震荡 | 区间交易 + FVG回补 | 资金费率极端反向 |
| 高波动 | 清算图猎杀 + 短线 scalp | 美股开盘联动 |
| 事件驱动 | 新闻交易 + 突破策略 | 链上大额异动 |

## 实际应用流程

### 每次交易前Checklist
1. [ ] **结构**: HTF方向？LTF结构突破？
2. [ ] **清算**: 附近有无高杠杆密集区？
3. [ ] **费率**: 当前资金费率？是否极端？
4. [ ] **美股**: MSTR/纳指走势？与crypto相关性？
5. [ ] **新闻**: 有无重大事件/数据？
6. [ ] **时间**: 是否在killzone内？
7. [ ] **RR**: 风险回报比 ≥ 1:2？
8. [ ] **仓位**: 单笔 ≤ 本金2%？

### 复盘维度
每笔交易记录：
- 触发条件（哪些维度满足）
- 实际结果
- 哪些维度有效/失效
- 市场环境分类

## 持续优化
通过大量回测和实盘，统计：
- 哪些维度组合胜率最高
- 不同市场环境下的最佳策略
- 个人执行中的弱点
