# 个人交易体系框架

## 一、已学策略分类整理

### 1. 日内交易 (Day Trading)
- **剥头皮 (Scalping)** - 持仓几秒到几分钟
- **流动性掠夺 (Liquidity Grabs)** - 利用止损猎杀
- **订单流分析 (Order Flow)** - 看买卖盘深度
- **市场结构突破 (BOS/CHoCH)** - ICT概念

### 2. 波段交易 (Swing Trading)
- **供需区交易 (Supply/Demand)** - 关键支撑阻力
- **公允价值缺口 (FVG)** - 价格回补缺口
- **订单块 (Order Blocks)** - 机构订单区域

### 3. 趋势跟踪
- **移动平均线策略** - EMA交叉
- **趋势结构分析** - 高低点排列
- **多时间框架分析** - HTF定方向，LTF找入场

### 4. 高级策略
- **清算图分析** - 识别价格磁铁
- **资金费率套利** - 多空情绪极端时反向
- **美股-Crypto联动** - MSTR/纳指作为领先指标

## 三、模拟盘练习计划

### 第一阶段：单策略熟练 (2-4周)
- [ ] 选择1个策略深入研究
- [ ] 在TradingView上回测至少50笔交易
- [ ] 记录每笔交易的入场理由、止损、止盈
- [ ] 统计胜率、盈亏比、最大回撤

### 第二阶段：多策略组合 (2-4周)
- [ ] 选择2-3个互补策略
- [ ] 练习策略切换判断

## 四、交易Checklist (每次交易前必查)

- [ ] Higher Time Frame方向确认
