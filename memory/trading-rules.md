# 综合交易分析框架

## 核心原则
**下单前必须综合分析以下所有因素，不能只看单一指标**

---

## 一、大势分析（宏观环境）

### 1. 美股走势
- **纳斯达克**：科技股走势，与crypto相关性高
- **微策略MSTR**：BTC领先指标，MSTR跌→BTC跌→ETH跌（1-2小时滞后）
- **标普500**：整体市场情绪

### 2. 新闻面
- 地缘政治（战争、国际关系）
- 美联储政策（利率、CPI、非农数据）
- 行业动态（ETF、监管、技术升级）
- 重大事件（交易所被黑、项目方跑路）

### 3. 市场周期
- 牛市/熊市/震荡市
- 当前处于什么阶段

---

## 二、技术分析

### 1. 趋势判断
- **MA均线**：MA7 > MA25 > MA99 = 多头；反之为空头
- **MACD**：金叉买入，死叉卖出
- **价格位置**：相对高/低位

### 2. 支撑阻力位
- 前高/前低
- 整数关口
- 密集成交区

### 3. 动量指标
- **RSI**：>70超买（考虑做空），<30超卖（考虑做多）
- **背离**：价格与指标背离是强烈信号

### 4. 波动率
- **布林带**：收窄预示突破，开口趋势延续
- **ATR**：衡量波动幅度

---

## 三、市场结构数据

### 1. 清算地图（CoinGlass）
- **高密度清算区**：价格"磁铁"，会吸引价格过去
- **链式反应**：一个清算引发更多清算
- **使用场景**：
  - 在清算密集区上方/下方找入场点
  - 止损放在清算区之外
  - 止盈放在下一个清算区

### 2. 多空比
- **>1.5**：市场极度看涨，警惕回调
- **<0.7**：市场极度看跌，警惕反弹
- **极端值**：反向操作信号

### 3. 爆仓数据
- 24h爆仓金额
- 多空爆仓比例
- 爆仓多的一方容易被继续挤压

### 4. 持仓量（Open Interest）
- **上升+价格上升**：趋势确认
- **上升+价格横盘**：即将突破
- **下降+价格下降**：趋势衰竭

### 5. 资金费率
- **正费率（多头付空头）**：市场看涨，可以收资金费做空
- **负费率（空头付多头）**：市场看跌，可以收资金费做多
- **极端费率**：反向信号

---

## 四、交易策略

### 1. 趋势跟随
- **适用**：明确趋势市场
- **方法**：回调到均线/支撑买入，反弹到阻力卖出
- **止损**：前低/前高

### 2. 突破交易
- **适用**：盘整后突破
- **方法**：突破关键阻力做多，跌破关键支撑做空
- **确认**：成交量放大
- **假突破**：快速返回区间内，立即止损

### 3. 回调买入
- **适用**：强势上涨趋势
- **方法**：等回调到支撑位/均线买入
- **不追高**：已经涨了5%+不追

### 4. 刮头皮（Scalping）
- **持仓时间**：几分钟到几小时
- **目标**：小利润，多次交易
- **关键**：低手续费，快速执行
- **止损**：非常紧，0.5-1%

### 5. 网格交易
- **适用**：震荡市场
- **方法**：在区间内低买高卖
- **风险**：突破区间会单边亏损

### 6. 资金费率套利
- **正费率**：做空收资金费
- **负费率**：做多收资金费
- **注意**：方向性风险

---

## 五、风险管理

### 1. 仓位控制
- **单笔风险**：不超过本金1-2%
- **总仓位**：不超过本金20-30%
- **杠杆**：新手5-10倍，熟练后15-20倍

### 2. 止损策略
- **技术止损**：支撑/阻力位下方/上方
- **百分比止损**：入场价±2-3%
- **时间止损**：持仓过久无盈利，平仓

### 3. 止盈策略
- **分批止盈**：50%仓位在R1，50%在R2
- **移动止损**：盈利后上移止损到成本价
- **目标位**：下一个阻力/支撑位

## 六、下单前Checklist

### 必须全部确认才能下单：

- [ ] **大势**：美股、新闻、宏观环境支持吗？
- [ ] **趋势**：当前趋势方向明确吗？
- [ ] **位置**：在关键支撑/阻力位附近吗？
- [ ] **动量**：RSI是否超买/超卖？
- [ ] **清算**：清算地图显示什么？
- [ ] **情绪**：多空比是否极端？
- [ ] **资金费**：资金费率方向支持吗？
- [ ] **止损**：止损位明确吗？能承受吗？
- [ ] **止盈**：目标位合理吗？
- [ ] **仓位**：杠杆和仓位合适吗？
